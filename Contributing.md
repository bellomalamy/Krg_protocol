# Contributing to KRG Protocol

Thank you for your interest in contributing to the KRG Offline State Synchronization Protocol (OSSP).

## How to Contribute

1. Open an Issue to report bugs, propose features, or suggest improvements.
2. Submit a Pull Request with clear documentation of your changes.
3. Follow the repository structure and do not break existing drafts.
4. All contributions must preserve the core principles:
   - Offline state propagation
   - Offline trust integrity
   - State reconciliation
   - Multi-hop relay logic

## Code of Conduct

Be respectful, constructive, and supportive.  
We are building something for the world.
